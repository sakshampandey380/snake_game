# 🐍 Snake Game Pro

A modern, responsive, and feature-rich Snake Game built using **HTML, CSS, and JavaScript**.
Designed with smooth animations, mobile controls, and a Play Store–style user experience.

---

## 🚀 Features

* 🎮 Classic Snake Gameplay with modern UI
* 📱 Fully Responsive (Mobile, Tablet, Desktop)
* 🕹️ Touch Controls + Swipe Support (Mobile Friendly)
* ⌨️ Keyboard Controls (Arrow Keys)
* 💾 High Score saved using Local Storage
* ⚡ Dynamic Speed Increase (Progressive Difficulty)
* 🎉 Win Popup after 20 points
* 💀 Game Over Popup with Score
* 🔊 Sound Effects (Eat & Game Over)
* 📳 Vibration Feedback (Supported Devices)
* 🖥️ Fullscreen Mode Support
* ✨ Glassmorphism UI + Glow Effects

---

## 🛠️ Tech Stack

* **HTML5** → Structure
* **CSS3** → Styling & Responsive Design
* **JavaScript (Vanilla)** → Game Logic & Interactions

## 🎮 How to Play

### 🖥️ Desktop:

* Use **Arrow Keys** to control the snake

### 📱 Mobile:

* Use **On-screen buttons**
* Or **Swipe gestures**

---

## 🧠 Game Rules

* Eat food to grow the snake 🍎
* Avoid hitting:

  * Walls ❌
  * Your own body ❌
* Reach **20 points to win 🎉**
* Game speed increases as you score more 🔥
